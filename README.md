# List Commands Collection

## WHAT

### What is **lscc**?

**lscc** is a improved version of some *ls* commands. </br>
E. g. `ls`, `lsusb`, `lsattr`, `lsmem`. 

### What for?

**lscc** is made for optimization of monitoring lists of system information with *manual*/*auto* updating.
